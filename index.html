REFUGEE RESTART

Refugee Restart is a multi-tenant application for use by charities that seek donations to help arriving refugees. The application allows users to search for families using various criteria and pledge donations by adding items to a cart. It is also configured to allow users to contribute to micro-loans requested by charities on behalf of refugees.
